商品管理 · 用户咨询 / 一对一回复

本页为静态页，无需 npm run build。适合在商品管理场景下：先看商品与发布者，再选商家加载该商家的用户咨询并回复。
说明：数据库里商品「发布者」来自用户表 yonghuId；咨询记录里「回答方」为商家表 shangjiaId。若两者在本业务中不是同一套 ID，请在下方手动选择商家，或使用 全量咨询列表。

① 商品列表（与后台「商品」同一接口）

点击一行可查看发布者信息（橙色高亮）。不选商品也可直接在下方选商家加载咨询。
